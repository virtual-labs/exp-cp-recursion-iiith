Add Numbers Factorial of Number Fibonacci series

Recursion Experiment (Computer Programming Lab)

Objectives

  • To understand that some problems can be broken down into smaller similar problems.
  • To solve such problems using recursive procedures.
    • To understand the concept of recursion.

Problem Statements

  1. Add two numbers using recursion.
  2. Find the factorial of a number using recursion.
  3. Find the nth Fibonacci number using recursion.
